Basic Structure Tags

<!DOCTYPE html>
Defines the document type and HTML version.

It tells the browser that the document follows HTML5 standards.

<!DOCTYPE html>
<html>
Root element of an HTML document.

All other elements of the web page are nested inside the <html> tag.

<html lang="en">
  <head>...</head>
  <body>...</body>
</html>
<head>
Container for metadata and document settings.

The <head> tag includes metadata, links to stylesheets, scripts, and the page title.

<head>
  <title>My Page</title>
  <meta charset="UTF-8">
  <link rel="stylesheet" href="style.css">
</head>
<title>
Specifies the title of the web page shown in the browser tab.

The <title> element helps identify the page and is important for SEO and accessibility.

<head>
  <title>My Portfolio</title>
</head>
<body>
Visible page content container.

All the content that appears on the webpage is placed inside the <body> tag, including text, images, links, and other elements.

<body>
  <h1>Hello, World!</h1>
  <p>Welcome to my website.</p>
</body>

üìÑ Metadata & Head Tags

<meta>
Provides metadata about the HTML document.

The <meta> tag gives information like character encoding, author, description, and viewport settings for responsive design.

<meta charset="UTF-8">
<meta name="description" content="Free HTML tutorial">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link>
Connects external resources like CSS or icons to the document.

The <link> tag is used inside the <head> to include external stylesheets, icons, or other linked files.

<link rel="stylesheet" href="style.css">
<link rel="icon" href="favicon.png" type="image/png">
<style>
Defines internal CSS styles for the document.

The <style> tag is used within the <head> to include CSS rules that style the HTML elements of the page.

<style>
  body { background-color: #111; color: #fff; }
  h1 { text-align: center; }
</style>
<script>
Embeds or links JavaScript code in an HTML document.

The <script> tag is used to add interactivity or functionality to a web page, either with inline code or an external JS file.

<script>
  console.log('Hello from JavaScript!');
</script>

<script src="app.js"></script>
<base>
Specifies the base URL for all relative links in a document.

The <base> tag sets a base path so that all relative URLs (like images, links, or scripts) resolve from it. It must appear inside the <head> and only once per page.

<base href="https://example.com/" target="_blank">
<a href="contact.html">Contact Us</a>

üß± Structural & Semantic HTML5 Tags

<header>
Represents the top section or introductory content of a page or section.

The <header> element usually contains a logo, navigation menu, heading, or other introductory elements for a webpage or article.

<header>
  <h1>My Website</h1>
  <nav>
    <a href="#">Home</a>
    <a href="#">About</a>
  </nav>
</header>
<nav>
Defines a navigation section with links to other pages or sections.

The <nav> element is used to group major navigation links, such as menus, tables of contents, or site navigation bars.

<nav>
  <a href="#home">Home</a>
  <a href="#about">About</a>
  <a href="#contact">Contact</a>
</nav>
<main>
Represents the main or central content of a web page.

The <main> element contains the unique content of the page ‚Äî the part that directly relates to the document‚Äôs primary purpose, excluding headers, footers, and sidebars.

<main>
  <article>
    <h2>Welcome</h2>
    <p>This is the main section of the page.</p>
  </article>
</main>
<section>
Defines a standalone section of related content.

The <section> element groups content into thematic sections, such as chapters, topic blocks, or tab panels, each usually with a heading.

<section>
  <h2>About Us</h2>
  <p>We build modern web solutions using the MERN stack.</p>
</section>
<article>
Represents an independent, self-contained piece of content.

The <article> element is used for standalone content such as blog posts, news articles, forum posts, or user comments that can be distributed independently.

<article>
  <h2>New Web Trends 2025</h2>
  <p>Developers are focusing on AI-powered web apps and improved accessibility.</p>
</article>
<aside>
Represents content indirectly related to the main content.

The <aside> element is often used for sidebars, pull quotes, ads, or related links that complement the main content but are not essential to it.

<aside>
  <h3>Related Articles</h3>
  <ul>
    <li><a href="#">Learn React</a></li>
    <li><a href="#">Intro to Node.js</a></li>
  </ul>
</aside>
<footer>
Defines the footer for a page or a section.

The <footer> element usually contains author information, copyright details, links, or contact info. It can be used for the entire page or individual sections.

<footer>
  <p>© 2025 MyWebsite. All rights reserved.</p>
  <a href="#top">Back to top</a>
</footer>
<figure>
Represents self-contained content like images, illustrations, or code snippets.

The <figure> element is used to group visual media and their captions. It helps semantically associate content like images or diagrams with descriptions.

<figure>
  <img src="mountain.jpg" alt="Mountain view">
  <figcaption>A scenic view of the mountains at sunset.</figcaption>
</figure>
<figcaption>
Provides a caption or description for a <figure> element.

The <figcaption> tag describes the content of a <figure>, such as an image, chart, or diagram. It can appear before or after the content inside <figure>.

<figure>
  <img src="sunrise.jpg" alt="Sunrise">
  <figcaption>Sunrise over the ocean.</figcaption>
</figure>
<details>
Creates a collapsible content section that can be expanded or hidden by the user.

The <details> element is often used for FAQs, extra information, or expandable sections. It works natively without JavaScript.

<details>
  <summary>What is HTML?</summary>
  <p>HTML stands for HyperText Markup Language.</p>
</details>
<summary>
Defines a visible heading for the <details> element.

The <summary> tag acts as the clickable title for a <details> section. When clicked, it expands or collapses the hidden content.

<details>
  <summary>Show more information</summary>
  <p>Here is the hidden content revealed on click.</p>
</details>
<dialog>
Defines a dialog box or popup window.

The <dialog> tag creates a modal or popup box that can be opened or closed using JavaScript. It‚Äôs great for alerts, forms, or messages.

<dialog id="myDialog">
  <p>This is a dialog box!</p>
  <button onclick="document.getElementById('myDialog').close()">Close</button>
</dialog>

<button onclick="document.getElementById('myDialog').showModal()">Open Dialog</button>
<mark>
Highlights or marks text for reference or emphasis.

The tag is used to highlight text, often for search results or important notes.

<p>This is an <mark>important</mark> word.</p>
<time>
Represents a specific time or date.

The tag is used to represent a specific time, date, or both ‚Äî useful for events or articles.

<p>Published on <time datetime="2025-11-02">November 2, 2025</time>.</p>

Deprecated Tags
Old HTML elements removed or discouraged in HTML5.

These tags were used in older versions of HTML. Modern web design uses CSS or newer semantic elements instead.

Old Tag Modern Replacement
<center> Use CSS: text-align: center;
<font> Use CSS: font-family, font-size, color
<big> / <small> Use CSS: font-size
<u> Use CSS: text-decoration: underline;
<marquee> Use CSS animations or JavaScript
<bgsound> Use <audio>
